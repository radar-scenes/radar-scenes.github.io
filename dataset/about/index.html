    The RadarScenes data set contains data from four automotive radar sensors mounted on one measurement vehicle. The data set was recorded between 2016 and 2018 in Ulm, Germany.

    Download

    Several publications from the machine learning field make use of the data set.

    A pre-print of the RadarScenes paper can be found on ArXiv.org.
